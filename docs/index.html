🚀 Start Audit

The auditor runs as a GitHub Actions workflow. Follow these steps:

  1. Go to the GitHub Actions page for this repository.
  2. Click GitHub Repo AI Auditor workflow on the left.
  3. Click Run workflow, enter the repo URL, and click Run.
  4. Wait ~5–10 minutes for the analysis to complete.
  5. The report will be published to this GitHub Pages site.
